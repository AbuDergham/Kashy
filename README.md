## ✨ الميزات الرئيسية

### 🛒 نقطة البيع
- شاشة بيع سريعة بدعم لوحة المفاتيح + شاشة لمس
- باركود + ماسح ضوئي
- **قراءة باركود الميزان الإلكتروني** (DIGI, Bizerba, CAS, Mettler-Toledo، والمزيد)
- طرق دفع متعددة: نقدي، بطاقة، تحويل بنكي، آجل
- تعليق الفاتورة (Hold/Park sale) لخدمة زبون آخر بسرعة
- إعادة طباعة آخر فاتورة بنقرة
- شبكة المنتجات الأكثر مبيعاً (Quick Pick)
- خصومات سريعة مع تأكيد للخصومات الكبيرة

### 📦 المنتجات والمخزون
- إدارة منتجات مع وحدات متعددة (base + sub units)
- باركود + رموز إضافية لكل منتج
- **تتبع تواريخ الانتهاء** مع تنبيهات FEFO
- **حد أدنى للسعر** (يمنع البيع تحت التكلفة)
- تتبع التكلفة والسعر للحصول على تقارير الربح
- تعديل أسعار بالجملة (Bulk price editing)

### 👥 الزبائن والموردين
- قاعدة بيانات شاملة مع كشوف حساب
- **مبيعات آجلة** مع حد ائتماني
- متابعة المعاملات والمدفوعات
- طباعة كشوف الحساب
- **إرسال كشف الحساب عبر WhatsApp** بنقرة 📱

### 📊 التقارير
- تقرير الحركة اليومية (مبيعات + ربح + إحصائيات)
- تقارير المنتجات الأكثر مبيعاً
- تحليلات بصرية بـ Chart.js
- المخزون المنخفض / المنتجات قاربت الانتهاء

### 🖥️ شاشة عرض الزبون
- شاشة منفصلة تظهر للزبون أثناء البيع
- **قابلة للتفعيل/التعطيل** (للمتاجر بشاشة واحدة)
- تخصيص كامل: خط، حجم، كثافة، animations، ألوان

### 💾 الأمان والنسخ الاحتياطي
- **نسخة احتياطية تلقائية قبل أي تحديث** (يحمي بياناتك)
- نسخ احتياطي محلي + Google Drive
- استعادة بنقرة من أي نسخة احتياطية
- فحص سلامة البيانات عند البدء
- **تحديث تلقائي** عند نزول إصدار جديد

## 📦 التثبيت

### تثبيت جديد

1. حمّل **[`Kashy-Setup-X.X.X.exe`](https://github.com/AbuDergham/Kashy/releases/latest)**
2. شغّل الملف (انقر نقرتين)
3. اتبع التعليمات
4. افتح التطبيق من قائمة Start أو سطح المكتب
5. سجّل دخول بالحساب الافتراضي:
   - **اسم المستخدم:** `admin`
   - **كلمة المرور:** `admin123`
6. **غيّر كلمة المرور فوراً** من صفحة الملف الشخصي

### الترقية من إصدار سابق

> ⚠️ **مهم:** بياناتك ستنتقل تلقائياً، لكن يُنصح بإنشاء نسخة احتياطية يدوية قبل الترقية.

1. التطبيق سيكتشف التحديث ويُنزّله تلقائياً (خلال ساعة)
2. **سيتم إنشاء نسخة احتياطية تلقائياً قبل التثبيت** (`%APPDATA%\kashy\pre-update-backups\`)
3. أعد تشغيل التطبيق عند طلب التثبيت
4. التطبيق يفتح بالإصدار الجديد مع كل بياناتك سليمة

### الاستعادة عند الحاجة

إذا حدثت مشكلة بعد تحديث:
1. افتح التطبيق
2. اذهب إلى **الإعدادات ← النسخ الاحتياطي**
3. اضغط **"استعادة من نسخة قبل التحديث"**
4. اختر أحدث نسخة من القائمة
5. التطبيق سيعود إلى الحالة قبل التحديث
